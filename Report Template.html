OncoKB: OncoKB database annotation and link to variant classification.

CIViC: CIViC database annotation and link to variant classification.

1.3. STRING Pathway Analysis

1.3.1. Network

The STRING database collects, scores and integrates all publicly available sources of protein-protein interaction information, and complements these with computational predictions. Its goal is to achieve a comprehensive and objective global network, including direct (physical) as well as indirect (functional) interactions. 3.1. Statistics

The PI_HAT (a measure of the genetic relatedness between two samples) between the germline and the somatic is PIKACHU_pihat, which means that the germline and the tumour PIKACHU_relatedness from the same individual. Tumour and germline samples from the same individual usually presnt a PI_HAT greater than 0.50.

Summary stats of the germline and somatic samples can be found in the table below.

PCRDup %: Percentage of the total reads that are PCR duplicates (fileterd out).

Average Coverage: Average coverage in the PIKACHU_Nregions regions sequenced that pass all QCs.

Mismatch %: The percentage of mismatches (i.e. point genetic variants) between all base pairs sequenced and the reference genome.

InDel %: The percentage of insertion and deletions between all reads sequenced adn the reference genome.

Strand Blance: the percentage of forward reads versus the total number of reads. Ideally, 50%.

Chimeras %: The percentage of pairs of reads that do not map to the same region in the correct orientation.

Adapter %: percentage of adaptor contamination in the reads sequenced.

4. Methods

DNA was extracted from buffy coats (for germline sample) or fresh frozen tumour/FFPE specimen (somatic sample) using the Qiagen DNA extraction kits (QIAsymphony DNA Mini Kit, AllPrep DNA/RNA FFPE Kit and QIAamp DNA FFPE Tissue kit). Libraries were prepared using a custom gene panel (Total Size: PIKACHU_panelsize base pairs) from NimbelGen ROCHE using the Roche Hyperprep kit and were sequenced on Illumina HiSeq4000. Sequencing data was analysed using CMDL's GCP compliant pipeline (version 0.41) which utilizes the following main algorithms: bwa-mem (hg38 alt contig aware alignment), samtools (PCR Duplicates removal), GATK (Base Quality Score, Recalibration, InDel Realignment), GATK's Haplotype Caller (Germline Variant Calling), GATK's MuTect2 (Somatic Variant Calling), GATK (CNV Calling), Delly (Non-CNV SV Calling) and Annovar (Variant Annotation). The limit of detection for somatic variants is PIKACHU_vafthreshold VAF.
